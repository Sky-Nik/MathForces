Задача №81: Суммы цешек (2)

Вычислите суммы: (a) \(C_n^k + C_{k+1}^k + \ldots + C_n^k\);

(b) \(C_n^1 + 2 C_n^2 + 3 C_b^3 + 4 C_n^4 + \ldots\);

(c) \(C_n^1 + 2^2 C_n^2 + 3^2 C_n^3 + 4^2 C_n^4 + \ldots\);

(d) \((C_n^0)^2 + (C_n^1)^2 + (C_n^2)^2 + \ldots + (C_n^n)^2\);

(e) \(C_n^0 + C_{n-1}^1 + C_{n-2}^2 + \ldots\).
